## Ⅰ. 函数嵌套调用

所谓函数嵌套调用指的是一个函数里面又调用了另外一个函数。  

#### 1. 执行流程
如果函数A中，调用了另外一个函数B，那么先把函数B中的任务都执行完毕之后才会回到上次 函数A执行的位置

#### 2. 示例
```python
def func_b():
    print("func: b")


def func_a():
    print("-----1-----")
    func_b()
    print("-----2-----")


func_a()
```
执行结果
```
-----1-----
func: b
-----2-----
```
